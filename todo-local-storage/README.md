Todo App (localStorage)

A minimal, client-side Todo list application that stores todos in the browser's localStorage.

How to use
- Open todo-local-storage/index.html in your browser (double-click or serve with a static server).
- Add, edit (double-click), toggle, and delete todos. Completed items will be preserved across reloads.

Files
- index.html — the app UI
- style.css — basic styling
- app.js — application logic using localStorage

Notes
- This is intentionally small and dependency-free. For production use consider accessibility improvements, tests, and optional frameworks.
