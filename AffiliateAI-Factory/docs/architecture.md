# Architecture

AffiliateAI-Factory terdiri dari empat lapisan utama:
- Input layer: produk, audiens, dan target platform.
- Agent layer: AI agents yang memproses riset dan pembuatan konten.
- Data layer: database dan schema JSON.
- Delivery layer: workflow publishing dan analitik.

Arsitektur ini memudahkan pemisahan tanggung jawab antar modul.
