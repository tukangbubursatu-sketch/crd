# Developer Guide

## Setup
1. Salin `.env.example` ke `.env`.
2. Install dependency yang dibutuhkan.
3. Jalankan aplikasi atau workflow sesuai kebutuhan.

## Kontribusi
- Gunakan struktur folder yang sudah disepakati.
- Simpan prompt di folder prompts/.
- Sertakan dokumentasi untuk perubahan penting.
