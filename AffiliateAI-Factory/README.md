# AffiliateAI Factory

> Autonomous Multi-Agent AI System for Affiliate Marketing

---

# Overview

AffiliateAI Factory adalah sistem Multi-Agent AI yang mampu menjalankan bisnis affiliate marketing secara otomatis dari awal hingga akhir.

Sistem tidak hanya membuat video, tetapi melakukan seluruh proses bisnis:

- Mencari produk trending
- Analisis affiliate
- Riset produk
- Analisis customer
- Analisis kompetitor
- Menulis script
- Generate prompt AI
- Generate video
- Generate subtitle
- Generate thumbnail
- Optimasi SEO
- Upload otomatis
- Analisis performa
- Belajar dari hasil
- Mengoptimalkan video berikutnya

---
